# 第一章：项目概述与架构

## 1.1 项目简介

**OpenManus** 是一个开源的通用 AI Agent 框架，能够实现类似 Manus 的通用任务处理能力。项目基于 MetaGPT 团队开发，无需邀请码即可使用。

### 核心特性

- **多Agent支持**：支持通用Agent、数据分析Agent、浏览器控制Agent、MCP Agent等多种类型
- **MCP协议**：完整实现 Model Context Protocol，支持连接外部MCP Server
- **A2A协议**：支持 Agent to Agent 通信协议
- **浏览器自动化**：集成 browser-use 库，支持网页浏览、表单填写、内容提取等
- **多LLM后端**：支持 OpenAI、Claude、Azure、Bedrock、Ollama 等
- **沙箱隔离**：Docker沙箱支持代码安全执行

## 1.4 核心设计模式

### 1.4.1 工厂模式 (Factory Pattern)

```python
# app/flow/flow_factory.py
class FlowFactory:
    @staticmethod
    def create_flow(flow_type: FlowType, agents: Dict[str, BaseAgent]) -> BaseFlow:
        if flow_type == FlowType.PLANNING:
            return PlanningFlow(agents=agents)
        # ... 其他类型
```

### 1.4.2 单例模式 (Singleton Pattern)

```python
# app/config.py
class Config:
    _instance = None
    _lock = threading.Lock()

    def __new__(cls):
        if cls._instance is None:
            with cls._lock:
                if cls._instance is None:
                    cls._instance = super().__new__(cls)
        return cls._instance
```

### 1.4.3 模板方法模式 (Template Method Pattern)

```python
# app/agent/base.py
class BaseAgent:
    async def run(self, request):
        # 模板方法：定义执行骨架
        while self.current_step < self.max_steps:
            result = await self.step()  # 子类实现
```

### 1.4.4 策略模式 (Strategy Pattern)

```python
# app/schema.py
class ToolChoice(str, Enum):
    NONE = "none"      # 不使用工具
    AUTO = "auto"      # 自动选择
    REQUIRED = "required"  # 必须使用工具
```
