## 13.5 安装指南

### 方式一：pip 安装

```bash
# 克隆项目
git clone https://github.com/MetaGPT/MetaGPT.git
cd OpenManus

# 安装依赖
pip install -r requirements.txt

# 安装 Playwright 浏览器（可选）
playwright install chromium
```

### 方式二：开发模式

```bash
# 克隆项目
git clone https://github.com/MetaGPT/MetaGPT.git
cd OpenManus

# 安装依赖（可编辑模式）
pip install -e .

# 安装预提交钩子
pre-commit install
```

## 13.8 依赖管理最佳实践

### 使用虚拟环境

```bash
# 创建虚拟环境
python -m venv venv

# 激活
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows

# 安装依赖
pip install -r requirements.txt
```

### 依赖锁定

```bash
# 生成锁定文件
pip freeze > requirements.lock.txt

# 从锁定文件安装
pip install -r requirements.lock.txt
```

### 可选依赖

```bash
# 仅核心功能
pip install openai anthropic pydantic

# 完整功能（含浏览器）
pip install browser-use playwright

# 开发依赖
pip install pytest black ruff
```

## 13.9 常见问题

### Q: 安装 browser-use 失败？

```bash
# 确保已安装 Playwright 浏览器
playwright install chromium
```

### Q: Docker 沙箱无法启动？

```bash
# 检查 Docker 是否运行
docker ps

# 确保有足够权限
sudo usermod -aG docker $USER
```

### Q: MCP 连接失败？

```bash
# 检查 MCP 服务器是否运行
curl http://localhost:3000/health

# 检查网络连通性
telnet localhost 3000
```
